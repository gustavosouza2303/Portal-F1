> Este é um projeto acadêmico nascido pela minha paixão pelo automobilismo. Este projeto irá explorar o universo da Fórmula 1, desde suas raízes históricas até o grid atual.


## 💻 Pré-requisitos

Antes de começar, verifique se você atende aos seguintes requisitos:

- Possuir um navegador web atualizado (Google Chrome, Mozilla Firefox, Microsoft Edge ou similar).
- Possuir um sistema operacional capaz de executar um navegador moderno (Windows, Linux ou MacOS).
- ⚠️ **Aviso:** Projeto otimizado para a resolução **1920x1080**. Divergências visuais podem ocorrer em outras resoluções.
